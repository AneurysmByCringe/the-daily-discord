# the-daily-discord
Discord Bot for Discord Hack Week. Spits out news and weather information whenever you want.

## How It Works
Utilizes free APIs to push out news and weather information whenever you want.  
**News:** Powered by NewsAPI.org 
[NewsAPI](https://newsapi.org)  
**Weather:** Powered by OpenWeatherMap
[OpenWeatherMap](https://openweathermap.org)  
Also uses [discord.py](https://github.com/Rapptz/discord.py), [Python Requests library](https://2.python-requests.org/en/master/), and the [Python Client library for News API](https://github.com/mattlisiv/newsapi-python).

## Usage
- Weather: $weather 'City Name (For cities with spaces use '+')'
- News: $news '2-letter country code' 'news category'

## Other stuff
Make sure to not forget all the tokens you need for this thing to work. They are all functional within their free plans.
